## ğŸˆ Using the Components in Your Projects

### Option 1: Copy & Paste

The simplest way to use these components is to copy the component files directly into your project.

1. **Copy the Component Files**: From `src/components/` to your project's components directory.
2. **Install Required Dependencies**: Make sure to install any external libraries the components depend on.
3. **Import and Use**: Import the component into your files and use it as you would any other React component.

### Option 2: Git Submodules (For the Git-Savvy)

If you want to keep the components updated across multiple projects:

1. **Add as a Submodule**:

   ```bash
   git submodule add https://github.com/your-username/my-react-components.git src/components
   ```

2. **Update Submodule When Needed**:

   ```bash
   git submodule update --remote src/components
   ```
